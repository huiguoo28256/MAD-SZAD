# MAD-SZAD

**MAD-SZAD** is a clinical multimodal dataset for schizophrenia and Alzheimer's disease analysis in real doctor-patient interview scenarios.

## Overview

MAD-SZAD is designed to support research on:

- cross-disease classification
- multimodal learning under missing modalities
- question-slot-level missing analysis
- causal-knowledge-driven multimodal modeling
- interpretable clinical AI

The dataset includes multimodal information collected from clinical interview scenarios, such as:

- audio
- video
- structured symptom scales

## Tasks

The dataset can be used for the following tasks:

1. Schizophrenia vs. Alzheimer's disease classification
2. Robust multimodal learning under missing modalities
3. Incomplete-input modeling in clinical interview settings
4. Causal structure learning and interpretability analysis
